# Expense Tracker
A simple expense tracker web app built using HTML, CSS and JavaScript.
Features:
1. User Authentication
Sign Up
Login
Logout
Password encryption
2. Expense Management
Add expense
Edit expense
Delete expense
View all expenses
3. Expense Details
Amount
Category
Date
Description
Payment method (Cash, UPI, Card, etc.)  
4. Categories
Examples:
Food
Travel
Shopping
Entertainment
Education
Bills
Healthcare
Other
5. Dashboard
Show:
Total expenses
Total income
Current balance
Number of transactions 
6. Income Tracking
Allow users to add:
Salary
Pocket money
Freelancing income
Gifts
7. Budget Setting
Monthly budget
Category-wise budget
Remaining budget display
8. Charts & Analytics
Use:
Pie chart for category-wise spending
Bar chart for monthly spending
Line chart for spending trends
9. Search & Filters
Filter by:
Date range
Category
Amount range
Payment method
10. Recent Transactions
Display last 5–10 transactions on dashboard.
11. Spending Insights
Examples:
"Food spending increased by 25% this month."
"Shopping is your highest expense category."
12. Savings Goal Tracker
Set savings target
Progress bar
Expected completion date
13. Recurring Transactions
For:
Rent
Netflix subscription
Internet bill
EMI
